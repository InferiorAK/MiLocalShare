## Installation

- Clone this repository -
  ```
  git clone https://github.com/inferiorak/MiLocalShare
  ```

- Now go to cloned directory -
  ```
  $ cd MiLocalShare
  $ pip3 install -r requirements.txt
  $ clear
  $ python3 app.py -h
  ```
  <br>
